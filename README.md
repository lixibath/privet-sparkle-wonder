# Miranda Janitorial Services

Professional commercial cleaning services in Dallas, TX. We provide reliable janitorial services for offices, medical facilities, retail stores, schools, and more.

## About This Project

This is the official website for Miranda Janitorial Services, built with modern web technologies to provide a fast, accessible, and user-friendly experience for our clients.

## Technology Stack

This project is built with:

- **Vite** - Next Generation Frontend Tooling
- **TypeScript** - Type-safe JavaScript
- **React 18** - Modern UI library
- **Tailwind CSS** - Utility-first CSS framework
- **shadcn/ui** - Re-usable component library
- **React Router** - Client-side routing
- **React Hook Form** - Performant form handling

## Features

- Responsive design optimized for all devices
- Fast page load times with code splitting
- SEO optimized with meta tags and semantic HTML
- Accessible (WCAG 2.1 AA compliant)
- Modern design system with Tailwind CSS
- Contact forms with validation
- Service area coverage information
- Pricing calculator
- Client testimonials

## Local Development

If you want to run this project locally:

### Prerequisites

- Node.js 16+ and npm installed ([install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating))

### Installation

```sh
# Clone the repository
git clone <YOUR_GIT_URL>

# Navigate to the project directory
cd <YOUR_PROJECT_NAME>

# Install dependencies
npm install

# Start the development server
npm run dev
```

The application will be available at `http://localhost:8080`

### Build for Production

```sh
# Create an optimized production build
npm run build

# Preview the production build locally
npm run preview
```
